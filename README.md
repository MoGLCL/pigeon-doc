# Pigeon documentation

Choose the deployment variant that matches your machine/environment:

| Target Variant | Guide | GitHub Repository | Database | OpenWA |
|---|---|---|---|---|
| Windows development/local | [WINDOWS.md](WINDOWS.md) | [Pigeon-Windows-Local](https://github.com/MoGLCL/Pigeon-Windows-Local) | Local or remote PostgreSQL | Existing checkout can be managed by Pigeon |
| Linux server with Docker | [DOCKER-LINUX.md](DOCKER-LINUX.md) | [Pigeon-Linux-Docker](https://github.com/MoGLCL/Pigeon-Linux-Docker) | PostgreSQL container | External gateway recommended |
| Private/shared hosting | [PRIVATE-HOSTING.md](PRIVATE-HOSTING.md) | [Pigeon-Private-Hosting](https://github.com/MoGLCL/Pigeon-Private-Hosting) | Hosting PostgreSQL | External gateway required unless host supports browser processes |

Also read [INTEGRATIONS.md](INTEGRATIONS.md) and [OPERATIONS.md](OPERATIONS.md).
